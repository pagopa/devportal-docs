# Assistenza e segnalazioni

### Assistenza per anomalie nel front office o sulle API esposte da PDND Interoperabilità

* Utilizza il link **“Assistenza”** nella testata del front office.
* Verrai indirizzato a un **form** in cui indicare **descrizione del problema**, **contesto operativo** e **indirizzo e-mail** per il riscontro.

### Segnalazioni tecniche e proposte di miglioramento

* È possibile aprire una **issue** sulla [repository GitHub operativa](https://github.com/pagopa/pdnd-interop-frontend/issues), preferibilmente utilizzando i **template** messi a disposizione.
* Le **issue pubbliche** devono essere **prive di informazioni riservate**, ad esempio **client assertion**, credenziali o token; eventuali segnalazioni contenenti dati sensibili vengono **rimosse dagli amministratori**. Per materiali riservati, utilizza i **canali di assistenza** indicati sopra.
