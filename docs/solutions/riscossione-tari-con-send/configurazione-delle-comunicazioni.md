# Configurazione delle comunicazioni

Per garantire un'esperienza utente chiara ed efficace su app IO relativa alla riscossione TARI, è fondamentale curare la configurazione della comunicazione.

### 🧩 Nome, descrizione e contenuto

* Oggetto chiaro e conciso: Utilizza un oggetto del pagamento inequivocabile, come "Tassa sui rifiuti 2025" o "Accertamento TARI 2024".
* Abstract informativo: Usa il campo abstract per fornire contesto immediato (es. periodo di riferimento della tassa, breve spiegazione della natura dell'atto).
* Puoi inserire un riferimento al numero di pratica.

{% hint style="warning" %}
Ricorda di omettere dati personali (come nome, cognome, codice fiscale o targa) nei campi descrittivi.
{% endhint %}

{% hint style="success" %}
Consulta il [**Manuale dei Servizi di IO**](https://developer.pagopa.it/app-io/guides/modelli-servizi) per consigli sulla stesura del contenuto.
{% endhint %}

### 📬 Comunicazioni attese

L'ente deve predisporsi a gestire:

* La creazione e firma dell'atto
* La creazione e gestione dell'avviso di pagamento
* Il reperimento dei dati dei cittadini a cui inviare le comunicazioni

### 🚀 Azioni e pulsanti

Le azioni previste sono:

* Apertura della notifica.
* Visualizzazione dell'atto.
* Pagamento, se previsto.
