# Come configurare le notifiche per la tua utenza

Puoi personalizzare quali avvisi ricevere e attraverso la pagina di configurazione delle notifiche.&#x20;

### Step 1: Accedere ad Area Riservata

{% embed url="https://selfcare.pagopa.it" %}

### Step 2: Entrare nella configurazione

Entra nella sezione "Notifiche" e successivamente "Configura notifiche".

### Step 3: Seleziona il canale

Le notifiche che si vuole ricevere sono configurabili per il canale in-app ("Notifiche") e email di cortesia ("Email").

### Step 4: Abilita le notifiche

Le notifiche sono disabilitate di default. Se vuoi ricevere comunicazioni, abilita gli switch corrispondenti alle variazioni di tuo interesse.

All'interno di questa sezione troverai un elenco delle categorie di eventi (es. e-service, Richieste di fruizione, Deleghe). Per ogni gruppo di eventi puoi attivare o disattivare la ricezione delle comunicazioni per il canale desiderato.

{% hint style="info" %}
In alternativa alle notifiche per ogni variazione, è possibile ottenere un riassunto settimanale. Maggiori informazioni nel [tutorial dedicato](come-abilitare-la-ricezione-di-promemoria-e-novita-settimanali.md).
{% endhint %}
