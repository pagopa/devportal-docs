# ❓ Contenuti utili all'assistenza

Per fornire maggiori informazioni all'utente riguardanti SEND - Servizio Notifiche Digitali o in merito al servizio Messaggi di cortesia è possibile riprendere i contenuti del [Centro Assistenza Cittadino](https://assistenza.notifichedigitali.it/hc/it) di SEND che sono costantemente aggiornati.
