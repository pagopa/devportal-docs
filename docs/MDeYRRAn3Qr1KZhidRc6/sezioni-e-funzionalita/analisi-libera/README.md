# Analisi libera

La sezione "Analisi Libera" è progettata per offrire al Prestatore Aderente un ambiente di consultazione flessibile, consentendo la generazione di una vista tabellare completamente personalizzabile. L’utente può configurare l’analisi selezionando i valori di interesse dei campi disponibili, adattando l’esplorazione dei dati alle proprie esigenze analitiche in modo preciso ed efficiente. Le visualizzazioni si aggiornano dinamicamente in base alle selezioni effettuate dall’utente, riflettendo sia l’andamento complessivo delle transazioni (_Dettaglio Transazioni piattaforma pagoPA_), sia il dettaglio specifico delle transazioni riconducibili al singolo Prestatore Aderente (_Dettaglio Transazioni gestite_), garantendo un'analisi simultanea su entrambi i livelli di aggregazione.
