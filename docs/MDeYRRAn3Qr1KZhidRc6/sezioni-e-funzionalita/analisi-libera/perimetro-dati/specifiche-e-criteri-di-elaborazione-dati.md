# Specifiche e criteri  di elaborazione dati

Le metodologie di elaborazione dei dati adottano i criteri illustrati nei paragrafi 3.1.1.3 per l'analisi dei dati riferiti all'intero perimetro (_Dettaglio Transazioni piattaforma pagoPA_) e 3.2.1.3 per l'analisi delle transazioni gestite dal Prestatore Aderente (_Dettaglio Transazioni gestite_).
