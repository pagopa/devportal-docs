# Transazioni Prestatore Aderente

La sezione "Transazioni Prestatore Aderente" è dedicata all'analisi delle transazioni del singolo Prestatore Aderente, la cui ragione sociale sarà visualizzata in sostituzione del generico riferimento "_Prestatore Aderente_". All'interno di questa sezione, sono disponibili le stesse dimensioni di analisi e le medesime elaborazioni presenti nella sezione "Overview Transazioni", applicate specificamente ai dati transazionali del Prestatore Aderente di riferimento.
