# Introduzione

Il presente manuale intende illustrare il funzionamento del Servizio **pagoPA Insights** (“Servizio”), destinato esclusivamente ai Prestatori Aderenti. Il documento si propone di fornire agli utenti una panoramica completa delle sue funzionalità, delle analisi disponibili e dei dati esposti, descrivendone la metodologia e i criteri di elaborazione. I primi capitoli sono dedicati alla descrizione dei requisiti necessari e della procedura di adesione al Servizio.
