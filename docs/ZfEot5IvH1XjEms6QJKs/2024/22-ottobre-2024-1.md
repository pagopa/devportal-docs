# 30/06/2025

`Nuove funzionalità`&#x20;

### Deleghe

Le pubbliche amministrazioni possono delegare le operazioni amministrative relative all'erogazione o alla fruizione di e-service ad altre pubbliche amministrazioni. [Approfondisci](22-ottobre-2024-1.md#deleghe)

### Template e-service

Standardizza il processo di creazione di e-service per casi d'uso ricorrenti. Un creatore mette a disposizione il template, altri aderenti ne derivano le proprie istanze di e-service da erogare, coerenti con il template stesso. [Approfondisci](https://app.gitbook.com/s/b8HnYwaAzhxRFAZdZBXL/manuale-operativo/template-e-service)

### Amministratore client

L'obbligo di dichiarare un'amministratore che sia responsabile dei _client API Interop_. È necessario per garantire la responsabilità amministrativa delle operazioni di scrittura sulla piattaforma effettuate attraverso le API di PDND Interoperabilità. [Approfondisci](https://app.gitbook.com/s/b8HnYwaAzhxRFAZdZBXL/manuale-operativo/client-e-materiale-crittografico#creare-un-client-api-interop)

### API v. 2, primo rilascio

La seconda versione delle API di PDND Interoperabilità prevede che tutte le operazioni effettuabili attraverso la UI del back office siano effettuabili anche via API. Nel primo rilascio sono inclusi alcuni endpoint di lettura e scrittura. [Approfondisci](https://selfcare.interop.pagopa.it/m2m/v2-interface-specification.yaml)

### DPoP: Distributed Proof of Possession

Un meccanismo che garantisce un livello di protezione avanzata per il voucher rilasciato da PDND Interoperabilità. Rende un voucher PDND inutilizzabile se sottratto. [Approfondisci](https://app.gitbook.com/s/b8HnYwaAzhxRFAZdZBXL/manuale-operativo/utilizzare-i-voucher#protezione-avanzata-del-voucher-con-dpop)

### Nuovi custom claim nel voucher PDND

Vengono introdotti quattro nuovi claim custom nel voucher rilasciato ai fruitori da PDND Interoperabilità. Servono per agevolare gli erogatori nelle verifiche dei voucher e sono accompagnati da due best practice emerse nel tempo. La funzionalità viene rilasciata disabilitata. [Approfondisci](https://github.com/pagopa/pdnd-interop-frontend/issues/1300)
