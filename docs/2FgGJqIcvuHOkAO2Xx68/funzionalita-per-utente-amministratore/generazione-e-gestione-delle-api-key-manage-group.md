# 3️⃣ Generazione e gestione delle API key Manage Group

Accedendo all’Area Riservata e successivamente al Back-office del prodotto IO, un amministratore può generare una coppia di API Key per un gruppo, scegliendolo tra quelli esistenti e per i quali non ne sia stata già generata una.

In seguito, l’amministratore può modificare una API Key di un Gruppo:

* può ruotare le chiavi: la rotazione delle chiavi è una pratica di sicurezza fondamentale che consiste nel rinnovamento periodico delle credenziali di accesso; questa pratica non è solo una raccomandazione ma spesso un requisito di conformità;
* può modificare l’elenco degli indirizzi IP abilitati a usarle;
* può eliminare la coppia di chiavi per il gruppo.
