# Come caricare un elenco di prodotti

Questo tutorial spiega come caricare un file in formato **CSV** contenente un lotto di prodotti da sottoporre a validazione per l'inserimento nell'Elenco del Bonus Elettrodomestici.

### Prerequisiti

Prima di procedere con il caricamento, è fondamentale che il file `.csv` sia stato preparato seguendo scrupolosamente le indicazioni fornite. La struttura del file varia a seconda che i prodotti siano registrati o meno sulla banca dati europea EPREL.

Per i dettagli sulla compilazione, consultare la sezione [Specifiche del file CSV](../riferimenti-tecnici/specifiche-del-file-csv.md).

### Procedura

Per caricare un nuovo lotto di prodotti, seguire i seguenti passaggi:

#### **Step 1 - Accedere alla sezione di caricamento**&#x20;

Per caricare un nuovo lotto di prodotti, selezionare la voce **"Caricamento lotti"** dal Menu di navigazione

#### **Step 2 - Selezionare la tipologia di prodotti**&#x20;

Nella pagina, il Produttore dovrà specificare se il file che sta per caricare contiene:

* Prodotti **registrati** sulla banca dati EPREL.
* Prodotti **non registrati** sulla banca dati EPREL.

#### **Step 3 - Selezionare e caricare il file**&#x20;

Fare clic sul pulsante **"Scegli file"** per selezionare dal proprio computer il file `.csv` corrispondente alla tipologia scelta. Una volta selezionato, il nome del file apparirà a schermo.

* **Step 4 - Avviare il processo** Fare clic sul pulsante **"Carica"** per inviare il file al sistema. Un messaggio di conferma notificherà l'avvenuta presa in carico del file.

#### Prossimi Passi

Una volta caricato, il file viene elaborato dal sistema in modo asincrono. Il Produttore non riceve un esito immediato, ma può monitorare lo stato di avanzamento del processo.

Per verificare l'esito del caricamento e scaricare eventuali report di errore, il Produttore deve consultare la sezione "Storico Caricamenti", come descritto nel tutorial [Come consultare lo storico dei caricamenti](come-consultare-lo-storico-dei-caricamenti.md).
