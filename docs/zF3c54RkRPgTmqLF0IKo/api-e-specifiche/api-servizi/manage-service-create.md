# Manage Service: Create

## Descrizione

Questa API permette di creare un servizio.&#x20;

{% hint style="info" %}
Devi usare la nuova chiave [`manage`](../../funzionalita/pubblicare-un-servizio/chiave-manage.md) per la gestione dei servizi.&#x20;
{% endhint %}
