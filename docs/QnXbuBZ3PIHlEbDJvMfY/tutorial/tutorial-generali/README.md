# Tutorial Generali

### Contesto e Obiettivi

Questa sezione raccoglie una serie di guide dedicate alle funzionalità amministrative e trasversali della piattaforma, che non sono strettamente legate ai ruoli di Erogatore o Fruitore ma sono fondamentali per la corretta gestione del proprio ente. Le operazioni qui descritte riguardano principalmente la configurazione dell'accesso, la gestione degli utenti e l'utilizzo del sistema di deleghe.

L'obiettivo di questi tutorial è fornire agli utenti Amministratori le istruzioni step-by-step per svolgere le attività di base necessarie a garantire la sicurezza e l'efficienza operativa. Seguendo queste guide, sarai in grado di gestire chi può accedere alla piattaforma per conto del tuo ente e di configurare rapporti di delega con altre organizzazioni.
