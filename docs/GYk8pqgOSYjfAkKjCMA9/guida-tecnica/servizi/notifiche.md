# Notifiche

PDND interoperabilità invia solo le seguenti notifiche:

All'erogatore:

* Nuova richiesta di fruizione ricevuta
* Finalità sopra soglia ricevuta

Al fruitore:

* Nuova versione e-service pubblicata (per l'e-service al quale l'ente è iscritto
* Richiesta di fruizione inoltrata: attivazione o rifiuto
* Finalità: rifiuto
* Finalità: attivazione o rifiuto adeguamento piano
