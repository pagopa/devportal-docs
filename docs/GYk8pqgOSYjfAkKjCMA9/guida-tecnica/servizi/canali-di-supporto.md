# Canali di supporto

## Per problemi ed errori che si riscontrano nel back office

Si troverà un link "Assistenza" nella testata di pagina, in alto a destra. Cliccando, si verrà rediretti a un form nel quale è possibile inserire i dettagli della propria richiesta e inserire una mail di contatto per ricevere risposta.&#x20;

## Per altre problematiche tecniche o proposte di migliorie

È possibile aprire una issue su [questa repository Github](https://github.com/pagopa/pdnd-interop-frontend/issues). Quando possibile, si prega di utilizzare i template messi a disposizione.

{% hint style="danger" %}
La client assertion è un'informazione sensibile. Non inserirla mai all'interno delle issue Github, accessibili pubblicamente. Ogni richiesta contenente informazioni riservate o sensibili sarà rimossa senza preavviso dagli amministratori.
{% endhint %}
