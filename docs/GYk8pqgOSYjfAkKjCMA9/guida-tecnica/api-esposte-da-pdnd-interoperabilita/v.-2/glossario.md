# Glossario

L'API è scritta nella lingua franca della programmazione, l'inglese. Di seguito si riporta la tabella di conversione dei termini in italiano.

Nelle pagine seguenti, quando si troverà un nome scritto con la prima iniziale maiuscola, si intenderà un oggetto di dominio; per un nome interamente in maiuscolo, si intenderà invece uno stato di un oggetto di dominio. In generale, laddove si trovino termini in inglese, si fa riferimento alla nomenclatura utilizzata per descrivere le API.

## Oggetti di dominio&#x20;

| Termine in API     | Significato            |
| ------------------ | ---------------------- |
| Agreement          | richiesta di fruizione |
| Attribute          | attributo              |
| Client             | client                 |
| Consumer           | ente fruitore          |
| Delegation         | delega                 |
| EService           | e-service              |
| EServiceDescriptor | versione di e-service  |
| EServiceTemplate   | template di e-service  |
| Key                | chiave pubblica        |
| Producer           | ente erogatore         |
| Purpose            | finalità               |
| PurposeVersion     | versione di finalità   |
| RiskAnalysis       | analisi del rischio    |
| Tenant             | ente aderente          |
| User               | utente                 |

## Stati

Gli oggetti di dominio possono assumere alcuni stati comuni. Stato qui si intende come "stato di una macchina a stati". Per comodità, si riporta un glossario di seguito, anche se non tutti gli oggetti implementano tutti gli stati e alcuni oggetti hanno degli stati particolari che non vengono indicati qui per brevità.&#x20;

Nelle pagine seguenti della guida sono comunque riportati i cicli di vita di ogni oggetto di dominio.

| Termine in API                   | Significato               |
| -------------------------------- | ------------------------- |
| ACTIVE o PUBLISHED               | attivo                    |
| ARCHIVED                         | archiviato                |
| DRAFT                            | in bozza                  |
| PENDING o WAITING\_FOR\_APPROVAL | in attesa di approvazione |
| REJECTED                         | rifiutato                 |
| SUSPENDED                        | sospeso                   |
