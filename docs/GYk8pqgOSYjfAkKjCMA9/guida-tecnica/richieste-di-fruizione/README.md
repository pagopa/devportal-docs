# Richieste di fruizione

Le richieste di fruizione sono quell'adempimento amministrativo che permette agli aderenti di iscriversi a fruire di un e-service presente nel Catalogo degli e-service di PDND Interoperabilità.

All'interno della richiesta, si verifica solamente che il fruitore possegga quei requisiti di accesso (chiamati attributi) che l'erogatore indica come condizione necessaria per l'iscrizione.

Proprio per questa ragione, è possibile avere una sola richiesta di fruizione attiva per ogni e-service.

In quanto adempimento amministrativo che autorizza l'ente ad estrarre dati dall'e-service di un erogatore, questa richiesta deve essere compilata e inoltrata da un'utenza con ruolo di amministratore.
