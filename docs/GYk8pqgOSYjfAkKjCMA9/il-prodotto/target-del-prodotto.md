# Target del prodotto

Attualmente possono aderire tutte le Pubbliche Amministrazioni, i gestori di servizi pubblici, le società controllate, le piattaforme di e-procurement certificate da AgID e le assicurazioni.

In futuro l'accesso sarà esteso alle imprese private come fruitori, le quali al momento possono accedere solo all'ambiente di Attestazione.

Per aderire, una PA deve innanzitutto essere accreditata sul Catalogo IPA. Successivamente, potrà seguire il percorso di adesione, firmando l'accordo di adesione alla piattaforma. Per ulteriori dettagli consulta la [guida all'adesione](https://docs.pagopa.it/interoperabilita-1/manuale-operativo/guida-alladesione#prima-delladesione).

Se la conformazione di un ente non permette di rientrare nei termini dell'accordo di adesione, è possibile contattare l'assistenza per valutare la situazione durante la procedura di adesione, attraverso il link all'assistenza in alto a destra.&#x20;
