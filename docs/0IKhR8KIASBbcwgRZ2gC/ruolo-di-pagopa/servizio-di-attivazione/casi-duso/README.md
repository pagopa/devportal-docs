# Casi d'uso

In questa sezione vengono illustrate le azioni  possibili   per il Service Provider.

Prenderemo in esame l'_attivazione_ del servizio per un singolo soggetto, piuttosto che la _rimozione_ del consenso a ricevere i flussi SRTP per un determinato soggetto ed un eventuale _subentro_ da parte  di un nuovo Service Provider per un determinato  soggetto.
