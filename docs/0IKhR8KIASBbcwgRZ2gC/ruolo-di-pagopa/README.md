# Ruolo di PagoPa

**PagoPA**  offrirà una serie di servizi a valore aggiunto per i Service Provider che vorranno interagire tramite lo schema SRTP di avvisi di pagamento pagoPA:&#x20;

* Servizio di **Attivazione \[Act],** progettato per facilitare la gestione delle attivazioni dei debitori per la ricezione delle notifiche degli avvisi di pagamento.  Questo servizio è particolarmente utile per tutti gli enti pubblici e i soggetti aderenti alla piattaforma, in quanto permette di ottimizzare il processo di notifica e di attivazione per i debitori, migliorando l’esperienza e l’efficacia del sistema di riscossione.
