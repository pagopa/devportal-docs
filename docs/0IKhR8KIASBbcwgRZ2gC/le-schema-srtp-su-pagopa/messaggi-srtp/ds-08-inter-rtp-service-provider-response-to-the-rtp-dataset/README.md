# DS-08 Inter-RTP Service Provider response to  the RTP Dataset

In caso di accettazione o rifiuto della SRTP, si considera il dataset “DS-08 Inter-RTP Service Provider Response to the RTP Dataset” del Rulebook SEPA Request To Pay, che prevede l’invio del messaggio di accettazione o rifiuto della SRTP da parte del Debitore. Il messaggio viene veicolato dal SRTP SP del Debitore al SRTP SP del Creditore.
