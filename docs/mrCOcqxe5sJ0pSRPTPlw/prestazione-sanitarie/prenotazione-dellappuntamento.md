# 2️⃣ Prenotazione dell'appuntamento

Il cittadino dopo aver ricevuto su IO il messaggio dell’emissione della prescrizione,  può prenotare la prestazione sanitaria cliccando sul link nel messaggio: per condurre un utente sull’area riservata di un sito terzo in modalità autenticata SPID/CIE è necessario integrare la funzionalità gratuita del Single Sign-On.
