# 📩 Invio delle richieste di pagamento



L' invio delle richieste di pagamento è la fase più delicata dell'esperienza: una **strategia comunicativa** ben definita e una **presentazione dei contenuti** ottimale sono fondamentali per instaurare un rapporto di fiducia con l'utente e massimizzare la conversione verso il pagamento.

{% hint style="warning" %}
Una richiesta di pagamento va intesa come **una delle modalità per comunicare all'utente dell'esistenza di un dovuto nei confronti di un ente.** Per questo motivo è importante considerare che l'utente deve poter approfondire, a partire da questa comunicazione, la natura del pagamento.
{% endhint %}
