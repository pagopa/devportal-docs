# v. 2

Come indicato, la v. 2 è la versione corrente dell'API di PDND Interoperabilità, e permette di operare sia in lettura che in scrittura, esattamente come farebbe un utente attraverso l'interfaccia grafica del back office.

Per operare in scrittura, sarà necessario nominare un amministratore dell'ente quale "amministratore del client".

Le API v. 2 sono in fase di rilascio. La versione più aggiornata della specifica è sempre consultabile [qui](https://developer.pagopa.it/pdnd-interoperabilita/api/pdnd-core-v2#/). Sono previsti rilasci a cadenza mensile fino ai primi mesi del 2026 per garantire la copertura di tutte le funzionalità. I dettagli degli endpoint rilasciati mano a mano sono comunicati nelle [Release note](https://developer.pagopa.it/pdnd-interoperabilita/release-note).
