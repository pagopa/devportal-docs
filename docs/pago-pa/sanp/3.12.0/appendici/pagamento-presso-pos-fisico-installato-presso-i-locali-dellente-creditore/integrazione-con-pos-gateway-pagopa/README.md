# Integrazione con POS Gateway pagoPA

Il POS Gateway pagoPA sarà una componente tecnologica messa a disposizione dalla piattaforma pagoPA per semplificare e standardizzare il dialogo tra il sistema gestionale in uso all'Ente Creditore e il terminale POS fisico. Questa componente ridurrà la complessità di implementazione a carico dell'Ente Creditore, garantendo al contempo che l'incasso locale sia nativamente integrato con i flussi di pagamento, riversamento e rendicontazione del nodo pagoPA.

Nelle sezioni successive vengono illustrate le modalità di integrazione con il POS Gateway pagoPA per EC e PSP finalizzate a gestire i pagamenti tramite POS fisico installato presso i locali dell'Ente Creditore:

* [Modalità integrazione EC](modalita-integrazione-ec.md)
* [Modalità integrazione PSP](modalita-integrazione-psp.md)
