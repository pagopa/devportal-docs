# Gestione Massiva

Al fine di facilitare l'inserimento delle posizioni debitorie relative agli Enti Creditori `EC` che aderiscono alla piattaforma `pagoPA` in modalità asincrona, sono state implementate le funzionalità di gestione massiva tramite:

* API REST
* SFTP.
