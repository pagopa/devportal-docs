# Elimina IBAN

La funzionalità **Elimina IBAN** è riservata esclusivamente al **Referente dei Pagamenti.**

{% hint style="danger" %}
**Nota:** Non è possibile eliminare un IBAN se questo è l'unico presente a sistema.
{% endhint %}

## Sezione Dettaglio IBAN: Elimina

All'interno della _**Pagina "Vista Iban"**,_ l'utente ha la possibilità di **cliccare** sulla **freccetta** blu accanto alla Data Scadenza per **accedere** alla pagina di **Dettaglio IBAN**.

<figure><img src="../../../.gitbook/assets/Screenshot 2026-01-08 alle 14.36.04.png" alt=""><figcaption></figcaption></figure>

All'interno di tale pagina è possibile **visualizzare** tutti i **dati** relativi all'**IBAN** inseriti in fase di creazione.

<figure><img src="../../../.gitbook/assets/Screenshot 2026-01-08 alle 14.36.18.png" alt=""><figcaption></figcaption></figure>

Da qui **cliccando** sull'apposito bottone **"Elimina"** l'utente ha la possibilità di definire la data in cui verrà effettuata la cancellazione effettiva dell'IBAN.

<figure><img src="../../../.gitbook/assets/Screenshot 2026-01-08 alle 14.36.26.png" alt=""><figcaption></figcaption></figure>

Scelta la data di eliminazione cliccare sul bottone **"Programma eliminazione"** per pianificare l'azione di cancellazione.&#x20;

<figure><img src="../../../.gitbook/assets/Screenshot 2026-01-08 alle 14.36.46.png" alt=""><figcaption></figcaption></figure>

Verrà nuovamente presentata la _**Pagina "Vista Iban".**_

<figure><img src="../../../.gitbook/assets/Screenshot 2026-01-08 alle 14.36.04.png" alt=""><figcaption></figcaption></figure>

**Cliccando** nuovamente sulla **freccetta** blu per **accedere** alla pagina di **Dettaglio IBAN** sarà possibile:

* Visualizzare la data di eliminazione pianificata
* Annullare la richiesta di cancellazione cliccando sul bottone "Annulla eliminazione"

<figure><img src="../../../.gitbook/assets/Screenshot 2026-01-08 alle 14.37.05.png" alt=""><figcaption></figcaption></figure>
