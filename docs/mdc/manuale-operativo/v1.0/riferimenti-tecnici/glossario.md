# Glossario

| Acronimo/Definizione                     | Descrizione                                                                                                                                                                                                                                                                                                                                                                                 |
| ---------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Comunicazioni a valore legale**        | Le comunicazioni a valore legale sono notifiche emesse da un'amministrazione mittente come multe, avvisi di accertamento di tributi o rimborsi. Le notifiche SEND sono quindi diverse dalle **notifiche push**, cioè quei brevi messaggi che ricevi sullo schermo del tuo dispositivo.                                                                                                      |
| **Avviso di avvenuta Ricezione**         | È il documento che ricevi via PEC o tramite serc-q di SEND o tramite raccomandata quando c’è una notifica per te. L'invio di questo documento ha valore legale perché contiene le informazioni essenziali della notifica                                                                                                                                                                    |
| **Avviso di Cortesia**                   | E' l’avviso in modalità informatica, contenente le stesse informazioni dell’avviso di avvenuta ricezione, inviato dal gestore della piattaforma, ai sensi dell’art. 26, commi 5-bis, 6 e 7, del decreto-legge n. 76 del 17 luglio 2020, come convertito, con modificazioni, dalla legge 11 settembre 2020, n. 120.                                                                          |
| **Terze Parti (TPP)**                    | Per Terze Parti si intende un soggetto che mette a disposizione dei propri Utenti dei touchpoint (app bancaria/web) per il Servizio “Messaggi di Cortesia”                                                                                                                                                                                                                                  |
| **Strong Customer Authentication (SCA)** | L'autenticazione forte del cliente (Strong Customer Authentication - SCA) è una procedura per convalidare l'identificazione di un utente basata sull'uso di due o più elementi di autenticazione, appartenenti ad almeno due categorie tra Possesso (qualcosa che solo l'utente possiede), Inerenza (qualcosa che caratterizza l'utente) e conoscenza (qualcosa che solo l'utente conosce). |
| **EMD**                                  | L'Enterprise Message Dispatcher è il layer di gestione dei messaggi avente lo scopo di armonizzare sia le interfacce esposte dalle piattaforme erogatrici, che le logiche di canale.                                                                                                                                                                                                        |
