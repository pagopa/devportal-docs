# Vincoli Operativi

Questa sezione elenca alcune limitazioni funzionali della piattaforma che l'Esercente (Venditore e Operatore PV) deve conoscere per una corretta gestione delle attività.

Si tratta di regole operative definite dal sistema che non possono essere derogate.

#### 1. Impossibilità di modifica dei Punti Vendita

Una volta che un Punto Vendita è stato registrato (censito) dal Venditore, **non è più possibile modificare i dati inseriti** (es. indirizzo o insegna). È fondamentale verificare la correttezza di tutte le informazioni prima della conferma finale.

#### 2. Assenza di caricamento massivo per i Punti Vendita

La piattaforma non supporta funzionalità di caricamento massivo (ad esempio tramite file CSV) per il censimento dei Punti Vendita. L'inserimento deve essere eseguito manualmente dal Portale Venditore, con un limite di **cinque Punti Vendita per singola operazione**.

#### 3. Gestione Password Operatore

Un Operatore PV non può resettare autonomamente la propria password. Se l'email di prima attivazione viene persa o il link scade, l'Operatore deve contattare il proprio **Venditore (Amministratore)**, il quale è l'unico autorizzato a **rinviare l'email di attivazione** dal Portale Venditore (come descritto nel tutorial [2.2 Creare e gestire gli Operatori](https://www.google.com/search?q=tutorial/creare-e-gestire-operatori)).
