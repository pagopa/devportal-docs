# Consultare l'Elenco informatico degli elettrodomestici

L'Operatore Punto Vendita avrà a disposizione la sezione "**Prodotti**" sul _Portale Esercenti_ grazie alla quale potrà consultare l'_Elenco informatico degli elettrodomestici_ facenti parte dell'iniziativa.

Tale elenco è soggetto a continui aggiornamenti.&#x20;

È possibile utilizzare la funzione di stampa che consente il download di un file contenente gli _Elettrodomestici_.
