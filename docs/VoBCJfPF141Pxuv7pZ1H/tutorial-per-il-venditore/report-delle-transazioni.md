# Report delle transazioni

Al fine di supportare il Venditore nelle attività di riconciliazione, è stata messa a disposizione una funzionalità che genera un report .csv contenente tutte le transazioni presenti in un dato intervallo temporale.

{% hint style="warning" %}
L'intervallo temporale non deve superare i 90 giorni e non può contenere la data odierna.
{% endhint %}

Dopo esser entrati nella sezione **Esporta report** sarà sufficiente selezionare le date relative all'intervallo temporale e cliccare sul pulsante **Genera report**.

La piattaforma mostrerà un popup in basso a destro che avvisa il Venditore circa la presa in carico della generazione del report. Inoltre, verrà visualizzata una riga contenente la generazione in corso del report che non sarà possibile scaricare fino al termine dell'elaborazione (la generazione può richiedere alcuni minuti).

Al termine dell'elaborazione, verrà mostrato un popup contenente l'esito dell'elaborazione del report e, sulla riga relativa allo stesso, sarà abilitata l'icona che permetterà il download del file .csv.
