* _\[Febbraio 2023]_ Integrata sezione [Verificare lo stato di una firma](https://developer.pagopa.it/it/firma-con-io/guides/manuale-operativo-fci/v1.0/verificare-lo-stato-di-una-firma) con le specifiche per ottenere la lista delle richieste di firma create, a partire da un Dossier.
