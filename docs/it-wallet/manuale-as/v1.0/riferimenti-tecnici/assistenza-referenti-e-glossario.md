# Assistenza, referenti e glossario

## Modello di assistenza

Nella sezione `assistenza` si indicano: **Referenti** (almeno 3 contatti per assistenza utenti, applicativo, sistemistico); **Canali** (almeno un canale di responsabilità dell'Ente); **FAQ**; **Testi informativi (opzionale)** (\~300–450 caratteri) da esporre prima dell'ottenimento dell'EAA.
